# 分析方法文档编写参考

执行 **SOP→分析方法** 前，请先阅读同仓库 **`SKILL.md` 文首「执行铁律」**；运行 `integrate_sop_method.py` / `refine_extracted.py` 时控制台会再次提示。

## LLM 整合规则（执行版）

读取 `extracted.json` 中每个方法的 `sop_raw`，参照 `reference_style` 的风格，对比整合并输出 `refined` 字段（不是自由改写）。

| 章节 | 精简要求 |
|------|----------|
| principle（原理） | **直接沿用** SOP 第四章「实验原理」**原文**（整段）；**禁止**概括、缩句、改写；仅可去除行首「原理：」前缀。无 SOP 时回退 `template_existing` / `reference_style`。 |
| materials_and_equipment（设备、材料、试剂） | 从 SOP 提取**全部**材料/设备/试剂的主要信息，**按类**写入标准文件（如「标准物质：…」「材料：…」「试剂：…」或「主要试剂/耗材：…」「主要设备：…」等与模版、`reference_style` 一致的前缀）；同类可多条，勿混成无标签长段。去除厂家/型号，保留试剂名称与关键规格。**特例**：方法章节为 **「纯度（SEC-HPLC）」** 时，「主要设备」**仅**输出一行「主要设备：高效液相色谱仪。」；该类仍不计入“省略设备枚举”的其它章节规则。 |
| sample_prep（样品处理） | 保留样品制备的关键流程，去除具体操作手法细节 |
| procedure（操作步骤） | 保留色谱条件参数、关键操作节点，去除仪器界面操作描述 |
| suitability_criteria（试验成立标准） | 保留判定指标和限度值，去除计算过程 |
| result_calculation（结果计算） | 保留核心公式和参数定义，去除仪器操作描述 |
| acceptance_criteria（合格标准） | 保留限度值，简化表述为一句话 |

**通用规则：**
- 保留项目编号（如 `CB1024-1`），不再主动删除
- 去除 SOP 编号引用（如 `3-SOP-QC-xxx`、`3-CB1024-xxx`）
- 去除公司内部流程引用和记录表编号
- 去除厂家名称（如 `杭州佑宁仪器有限公司`）
- 风格参照 `reference_style` 中对应章节的段落长度和措辞
- **可追溯性强约束：`refined` 中每一句都必须在 `sop_raw` 或 `template_existing` 中可定位到原文依据，禁止新增无来源信息**
- **禁止编造：禁止补充来源中不存在的新结论、新参数、新阈值、新公式**
- **优先级：SOP 当前内容 > template_existing > reference_style**
- **同义词归一化：允许术语归一，但不改动事实和数值**
  - 例如：`工作参比品/参比品/对照品` -> `标准物质`
  - 例如：`仪器/设备` -> `主要设备`
  - 例如：`可接受标准/判定标准` -> `合格标准`
- **标准物质映射规则（条件性）：仅当 SOP 或模板原文存在标准物质/参比品相关信息时，才输出 `标准物质`；且优先对应 SOP 中“系统适用性样品/系统适用性溶液”对应的参比品内容**
- **归一化边界：仅允许术语归一和语序微调，数值、单位、比较符号（<=/>=/%/mg/mL 等）必须与来源一致**
- **冲突处理：若 SOP 与模板内容冲突，以 SOP 为准；不得混写两个版本**
- **设备、材料、试剂分类写全（materials_and_equipment）**：整合到申报稿时，须在 SOP 有据前提下写全各分类下的主要内容，再写入模版；**纯度（SEC-HPLC）** 章节的 **「主要设备」** 行固定为「主要设备：高效液相色谱仪。」（脚本写回阶段亦会强制），**标准物质 / 试剂（含流动相等）/ 材料** 等仍按 SOP 分类写全。
- **手工编写 `refined.json`**：`principle` 须与上表首行及 SOP「实验原理」原文一致；**不适用**下文「原理 100～200 字」概要口径。

## 各章节内容要求

**说明**：以下各小节中的「字数要求」（如操作步骤 100～200 字、合格标准 30 字以内等）主要供**方法概要 / 非流水线**写作参考。**SOP→分析方法** 正式流水线以 `refine_extracted.py` 的实现与上文「LLM 整合规则」表为准，实际篇幅随 SOP 与模板占位而异。

### 1. 原理

**SOP→模版整合**：以 SOP「实验原理」**原文**为准，**不限制**本节字数、句数（见上文 LLM 整合规则表）。

**方法概要/非整合场景可参考字数**: 100-200 字

**内容要求**（概要场景）:
- 基于通用科学原理，说明方法"为何成立"
- 描述被测对象与信号的本质关系
- 突出定量关系和线性关系
- 不强调仪器或技术名称

**正确示例**:
```
采用体积排阻高效液相色谱法测定样品纯度。根据分子大小差异，不同分子量的
蛋白质在色谱柱中的保留时间不同，大分子先流出，小分子后流出，从而实现
分离检测。通过紫外检测器在280nm波长处检测信号，采用面积归一化法计算
各组分含量百分比。
```

**错误示例**:
```
使用安捷伦1260液相色谱仪，配合TSK gel G3000SWXL色谱柱进行检测...
（出现了品牌名称，不符合要求）
```

### 2. 设备、材料、试剂

**篇幅**：以「分类列全 SOP 主要信息」为准，**不再**以 50 字为硬上限；语句仍应简练，避免堆砌操作细节。

**内容要求**:
- 界定方法成立所依赖的"必要条件边界"
- 使用类别化、概括性名称（不出品牌、型号、项目背景）
- **按类输出**：将 SOP 中的材料、设备、试剂**先归类**再写入（与模版段落前缀一致，如标准物质、试剂、主要设备、主要试剂/耗材等）

**纯度（SEC-HPLC）例外**：「主要设备」仅写 **高效液相色谱仪** 一行；其余类别仍按上款写全。

**分类格式（示例，以前缀与模版为准）**:
```
标准物质：…
材料：…（若有）
试剂：… 或 主要试剂/耗材：…
主要设备：…
```

**正确示例**:
```
设备：高效液相色谱仪、离心机、体积排阻色谱柱。
材料：工作参比品、样品、缓冲液。
试剂：流动相（100mM 磷酸盐缓冲液，150mM NaCl，pH 7.0±0.2）
```

### 3. 操作步骤

**字数要求**: 100-200 字

**内容要求**:
- 描述"获取原始数据的基本路径"
- 只保留关键动作节点
- 不出现"点击、调节、等待"等操作级语言
- 分为样品处理和检测操作两部分

**结构格式**:
```
样品处理：
- [溶液类型1]：[制备方法]
- [溶液类型2]：[制备方法]
- ...

操作步骤：
[简洁描述检测流程]
```

**正确示例**:
```
样品处理：
- 空白溶液：取缓冲液直接进样
- 系统适用性溶液：用缓冲液稀释工作参比品至约1.0 mg/mL
- 供试品溶液：用缓冲液稀释供试品至约1.0 mg/mL

操作步骤：
按色谱条件设置仪器参数，依次进样空白溶液、系统适用性溶液和供试品溶液，
记录色谱图，计算各组分含量。
```

### 4. 可接受标准/系统适用性

**字数要求**: 每条 30 字以内

**内容要求**:
- 回答"在什么条件下，结果才被承认为有效"
- 全部采用量化、可判定的指标
- 明确统计学或逻辑学判据

**格式**:
```
- [指标1]应[符号][数值]
- [指标2]应[符号][数值]
```

**常见指标**:
| 指标类型 | 示例 |
|----------|------|
| RSD | 连续5针峰面积RSD应≤2.0% |
| 保留时间 | 保留时间RSD应≤2.0% |
| 分离度 | 主峰与相邻峰分离度应≥1.5 |
| 线性 | 线性相关系数R²应≥0.999 |

### 5. 结果计算

**内容要求**:
- 定义从原始数据到最终结果的标准转换规则
- 以公式或规则表达为主
- 明确每个变量的含义与单位

**格式**:
```
[公式名称]：[数学公式]

其中：
[变量1]：[含义和单位]；
[变量2]：[含义和单位]；
...
```

**示例**:
```
使用面积归一法计算：
主峰% = 主成分峰面积 / 总蛋白峰面积 × 100%

其中：
主成分峰面积：主峰的积分面积；
总蛋白峰面积：所有蛋白相关峰面积之和。
```

### 6. 合格标准

**字数要求**: 30 字以内

**内容要求**:
- 给出结果的"合规边界"
- 使用明确数值区间或阈值
- 中性、裁判式表达

**格式**:
```
[指标名称][符号][数值/范围]
```

**示例**:
```
主峰%不低于95.0%
聚体%不高于5.0%
蛋白含量：117.0~143.0 mg/mL
```

### SEC-HPLC 纯度检验示例

```markdown
**原理**
采用体积排阻高效液相色谱法（SEC-HPLC）测定样品纯度。根据分子大小差异，
不同分子量的蛋白质在色谱柱中的保留时间不同，从而实现分离和定量检测。

**设备、材料、试剂**
设备：高效液相色谱仪、离心机、TOSOH TSK gel G3000PWXL色谱柱。
材料：工作参比品、样品、FB溶液。
试剂：流动相（100mM PB，150mM NaCl，pH7.0±0.2）

**操作步骤**
样品处理：
- 空白溶液：取FB溶液直接进样
- 系统适用性溶液：用FB稀释工作参比品至约1.0 mg/mL
- 供试品溶液：用FB稀释供试品至约1.0 mg/mL

操作步骤：
按色谱条件设置仪器参数，依次进样空白、系统适用性、供试品溶液进行检测。

**可接受标准**
- 系统适用性溶液连续5针主峰保留时间RSD应<=2.0%
- 系统适用性溶液连续5针主峰峰面积RSD应<=2.0%

**结果计算**
使用面积归一法计算：
主峰% = 主成分峰面积/总蛋白峰面积 × 100%
聚体% = 多聚体峰面积/总蛋白峰面积 × 100%
低分子量杂质% = 低分子量组分峰面积/总蛋白峰面积 × 100%

**合格标准**
主峰%不低于95.0%
聚体%不高于5.0%
```

### SoloVPE 蛋白含量输出示例

```markdown
1  原理：蛋白质分子中所含有的色氨酸、苯丙氨酸、二硫键等在280nm波长附近有最大吸收峰，其光吸收值与蛋白质浓度成正比。基于朗伯-比尔定律（Lambert-Beer Law），通过建立吸光度-光程的线性方程式而测得斜率k=ΔA/ΔL（A为光吸收值，L为光程），并由朗伯-比尔定律公式A=εLc（ε为消光系数）换算得到待测样品的浓度c。

2  材料和设备
2.1  材料：比色皿、光纤棒。
2.2  主要设备：可变光程紫外-可见分光光度计。

3  操作步骤
取供试品上下颠倒混匀后，待测。启动可变光程紫外-可见分光光度计，选择预设了波长为280 nm和320 nm、消光系数为1.6000 mL/(mg*cm)、基线校正的方法，插入光纤棒，取供试品溶液进行测定。

4  试验成立标准：
4.1  线性相关系数R2应>=0.999
4.2  每个供试品溶液重复三次蛋白含量的RSD应<=3.0%
4.3  两个平行样间的蛋白含量RSD应<=3.0%

5  结果计算
朗伯-比尔定律（Beer-Lambert law）：A=εL Conc.
Slope=ΔA/ΔL
Conc.=Slope/ε×10
其中：
Slope：吸光值与光程的线性回归斜率；
A：吸光度值；
L：光程，mm；
Conc.：供试品浓度，mg/mL；
ε：消光系数，CB1019蛋白的消光系数为1.6000 mL/(mg*cm)。

6  合格标准：117.0~143.0 mg/mL。
```
